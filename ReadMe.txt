Simple Java Messenger
_______________________________________

Запустите сервер. Для этого скопируйте путь к файлу messenger-1.0.0.jar расположенному в target. 
Далее введите в командную строку windows следующую комнду:

java -cp *path* messenger.target.Server

Для запуска клиента:

java -cp *path* messenger.Main

Введите свой логин. Для отправки сообщений перед текстом сообщения укажите имя получателя с двоеточием.
_______________________________________

To start the server, copy the path to the messenger-1.0.0.jar file from target.
Open windows powershell and enter the following:

java -cp *path* messenger.target.Server

To start the client:

java -cp *path* messenger.Main
Enter your login. Before sending the message, enter the receiver's login.
